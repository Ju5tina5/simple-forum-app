<!-- GETTING STARTED -->
## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ju5tina5/simple-forum-app.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start react development enviroment
   ```js
   npm start
   ```
